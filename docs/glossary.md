# Glossary

## CPU

Central Processing Unit. General-purpose processor that runs operating-system, application, and orchestration workloads.

## GPU

Graphics Processing Unit. Parallel accelerator commonly used for AI training, inference, simulation, and other compute-heavy workloads.

## PCIe

Peripheral Component Interconnect Express. High-speed interconnect used for GPUs, NICs, NVMe devices, and other expansion hardware.

## NUMA

Non-Uniform Memory Access. A system architecture where memory access latency depends on which CPU socket or memory controller owns the memory.

## RAM

Random Access Memory. Volatile system memory used by running processes and the operating system.

## ECC Memory

Error-Correcting Code memory. Memory that can detect and correct certain memory errors, important for server reliability.

## NVMe

Non-Volatile Memory Express. A high-performance storage protocol commonly used by SSDs over PCIe.

## NIC

Network Interface Card. Hardware interface used for Ethernet or other network connectivity.

## Kernel

Core part of the operating system that manages hardware, processes, memory, filesystems, and device drivers.

## Kernel Module

Loadable code that extends the Linux kernel, often used for device drivers.

## Driver

Software component that allows the operating system to communicate with hardware.

## Firmware

Low-level software running on hardware devices such as NICs, storage devices, GPUs, BIOS, or BMCs.

## BIOS

Basic Input/Output System. Platform firmware responsible for early hardware initialization and boot configuration.

## BMC

Baseboard Management Controller. Independent management controller used for out-of-band monitoring, power control, and sensor data on servers.

## CUDA

NVIDIA parallel computing platform and programming model used by many AI and HPC applications.

## NVIDIA-SMI

NVIDIA System Management Interface. Command-line tool for querying NVIDIA GPU state, driver version, utilization, temperature, power, and running processes.

## NVML

NVIDIA Management Library. Library used by tools and applications to query and manage NVIDIA GPU state.

## DCGM

NVIDIA Data Center GPU Manager. Tooling for GPU telemetry, diagnostics, and fleet management in data-center environments.

## Thermal Throttling

Automatic reduction in clock speed or performance to keep hardware within temperature limits.

## Power Limit

Configured or enforced maximum power draw for a device such as a GPU.

## Telemetry

Measurements collected from a system, such as temperature, power, utilization, errors, clocks, and fan speed.

## Validation

Process of confirming that a system meets expected hardware, software, reliability, and performance requirements.

## Stress Test

Workload designed to put controlled load on a system or component to observe behavior under pressure.

## Burn-In Test

Longer-duration test intended to expose stability, thermal, power, or component issues before production use.

## Root-Cause Analysis

Structured investigation to identify the underlying cause of an issue rather than only the immediate symptom.

## Escalation

Process of handing an issue to a higher support tier, engineering team, vendor, or hardware owner with enough evidence for action.
